# Purpose

The Business Question Library defines the analytical questions that the Retail Intelligence Platform is expected to answer.

Rather than performing analysis without direction, every SQL query, Python script, visualization, KPI, and Power BI report developed during this project must address one or more documented business questions.

This approach ensures that analytics remains business-driven and aligned with decision-making needs.

---

# Objectives

The objectives of this document are to:

* Capture business requirements.
* Organize analytical questions by business function.
* Support SQL development.
* Guide Python analysis.
* Drive dashboard design.
* Define reporting priorities.
* Ensure every KPI answers a real business problem.
